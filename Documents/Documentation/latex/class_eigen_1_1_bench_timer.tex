On P\+O\+S\+IX platforms we use clock\+\_\+gettime with C\+L\+O\+C\+K\+\_\+\+P\+R\+O\+C\+E\+S\+S\+\_\+\+C\+P\+U\+T\+I\+M\+E\+\_\+\+ID. On Windows we use Query\+Performance\+Counter
